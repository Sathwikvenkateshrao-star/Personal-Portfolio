#  Sathwik Venkatesh Rao – Personal Portfolio

A modern, animated, and conversion-focused personal portfolio built to showcase my work as a **Full-Stack Developer** specializing in **Web Applications, AI Systems, and Automation**.

This portfolio is designed not just to display projects, but to clearly communicate **how I work, what problems I solve, and why founders and teams should work with me**.

##  Features

-  **Next.js App Router** architecture
-  **Dark & Light Theme** with glassmorphism UI
-  **AI-focused sections** (LangChain & LangGraph expertise)
-  Custom cursor & smooth scrolling
-  Scroll-triggered and hover-based animations
-  Modular, scalable component structure
-  Fully responsive (mobile → desktop)
-  Conversion-oriented copy & layout (startup-friendly)
